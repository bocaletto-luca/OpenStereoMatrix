OpenStereoMatrix

A passive wiring method to adapt 2.0 stereo amplifiers for expanded speaker layouts including 3.0, 4.0, 5.0, 2.1, and 5.1.

Overview

This document provides standalone HTML diagrams and wiring guides for each mode supported by OpenStereoMatrix. No images, software, or external drawings are required — all logic is illustrated using basic SVG elements inside this file.

  • Compatible with analog stereo amplifiers, provided load impedance and ground architecture are respected.
  • Recommended only for setups where the amplifier’s minimum impedance is known and respected.
  • All connections use resistive matrices; no active electronics are required.

Mode: 2.0 (Direct Stereo)

Standard stereo connection. No additional wiring is necessary. Connect the amplifier’s left and right outputs directly to the respective front speakers.

Use high-quality speaker cable, observe polarity, and check amplifier specifications for safe load ranges.

Mode: 2.1 (Subwoofer Feed)

Derive a line-level signal from the amplifier’s speaker outputs via a high-impedance voltage divider. This feeds an active subwoofer that accepts high-level input or RCA line-in.

  • Each channel uses a 10 kΩ resistor in series with the signal, then a 1 kΩ resistor to ground at the RCA tip.
  • For mono input, sum L and R with two 10 kΩ resistors.
  • Subwoofer must handle line-level inputs derived from speaker terminals.

Resistors can be small (1/4 W). For safety, always confirm the amplifier’s negative terminals are referenced to ground. If using a class-D amp without common ground, the subwoofer must isolate the inputs.
